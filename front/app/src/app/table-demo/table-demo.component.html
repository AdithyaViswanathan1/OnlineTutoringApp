Time Table Demo
The time table component has 3 modes depending where it is used, which is controlled by an input variable
Past dates and the current day cannot be selected in all cases
0: View only
No selection allowed
1: Tutor edit schedule
Can select any tile to toggle it as being in the tutor's schedule
Deselecting an already booked slot will trigger a warning to be confirmed before saving
2: Student appointment booking
Can select only green tiles that are on the schedule and not already booked
